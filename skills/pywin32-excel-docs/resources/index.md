## Object Hierarchy

```
Application
  └─ Workbooks -> Workbook
       ├─ Worksheets -> Worksheet
       │    ├─ Range (Cells, Rows, Columns, ...)
       │    │    ├─ Font, Interior, Borders
       │    │    ├─ FormatConditions
       │    │    └─ Hyperlinks, Comments, Names
       │    ├─ Shapes -> Shape
       │    ├─ ListObjects -> ListObject
       │    │    ├─ ListColumns, ListRows
       │    ├─ PageSetup
```
